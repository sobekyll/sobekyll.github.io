你好Sobekyll

November 20

SobeKyll 是一个基于SoberJS开发的Jekyll Material Design风格的主题.

安装

在Github Pages中使用

因为Github Pages对Jekyll以及相关库和版本的支持都有限，默认只支持部分库以及有限的版本，所以在Github Pages中使用Sobekyll需要特别说明。相关信息请参考: Dependency versions

在你博客目录的根目录下，创建.github/workflows/jekyll.ymlaction配置文件，并添加如下代码：

name: Build and Deploy Jekyll

on:
  push:
    branches:
      - main  # 或者你的默认分支

permissions:
  contents: write  # 允许对仓库内容进行写操作

jobs:
  build:
    runs-on: ubuntu-latest

    steps:
    - name: Checkout Code
      uses: actions/checkout@v3

    - name: Setup Ruby
      uses: ruby/setup-ruby@v1
      with:
        ruby-version: '3.1' # 确保版本兼容 Jekyll

    - name: Install Dependencies
      run: |
        gem install bundler
        bundle install

    - name: Build Site
      run: bundle exec jekyll build

    - name: Deploy to GitHub Pages
      uses: peaceiris/actions-gh-pages@v4
      with:
        github_token: $
        publish_dir: ./_site

这里需要注意的是分支名称main，在部分早期创建的repo中，需要改为master，这取决于你项目的主分支是什么。

on:
  push:
    branches:
      - master  # 早期项目需要改为master

在Gemfile中，添加gem 'sobekyll'并执行bundle install，然后在_config.yml文件中配置主题：

theme: sobekyll

设置完成，将修改推送到GitHub，并在repo的Settings -> Pages, 将Build and deployment 项改为Deploy from a branch, 并把分支改为’gh-pages’。修改后需要等待一段时间生效。
